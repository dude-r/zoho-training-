# Screenshots

Drop screenshot images here and reference them from the guides.

Each guide currently uses text placeholders like `[Screenshot: Navigate to Accounts tab]`. When you capture the real image:

1. Save it in this folder with a descriptive name, e.g. `accounts-new-button.png`.
2. In the guide, replace the placeholder with:
   ```markdown
   ![Navigate to Accounts tab](assets/accounts-new-button.png)
   ```

Keep names lowercase with hyphens, and prefer PNG for UI screenshots.
