# Screenshot Shot List

Every `[Screenshot: …]` placeholder in the guides, turned into a capture checklist. Work down it with Zoho (and Outlook) open, save each image into this `guides/assets/` folder using the **filename** given, then replace the matching placeholder in the guide with:

```markdown
![short description](assets/the-filename.png)
```

Naming follows the convention in [README.md](README.md): lowercase, hyphenated, PNG. **52 shots total.** Tick them off as you go.

> Tip: capture at a consistent width, crop tight to the relevant control, and blur any real client names/PII before saving.
